COMBO DEMONSTRATION IMAGES
===========================

This directory contains full-combo sequence screenshots showing complete Vi combos.

Required Images:

1. vi-combo-structure.png
   - Diagram or infographic showing Vi's combo flow
   - Sequence: Starter → Launcher → Rekka → OTG → Super Finish
   - This can be text-based diagram or visual flowchart
   - Used in Combo Guide chapter header

2. combo-beginner-bnb.png
   - Visual demonstration of: 5M > 5H > 6S1~L~M~H > 2S2
   - Show the full sequence of hits
   - Should display damage values and positions
   - Good for teaching fundamental combo structure

3. combo-limit-strike.png
   - Visual demonstration of: 5M > 5H > 5S2 > dash > 5M > 5H > 2H > j.S1
   - Shows the dash restart and wall slam end
   - Demonstrates limit strike concept
   - Can show multiple frames of sequence

Optional Additions:
- combo-wallbounce.png - Wall bounce examples
- combo-oki-setup.png - Okizeme setup scenarios
- combo-corner-loop.png - Corner-specific combos
- electric-combo.png - Electric version combos with more damage

How to Create Combo Breakdowns:

Option 1: Sequential Screenshots
- Record combo inputs in training mode
- Take screenshot at each major hit
- Use image editor to tile/arrange them
- Label each step with inputs and damage

Option 2: Single Screenshot
- Capture at peak moment of combo
- Should show multiple characters/effects
- Or create infographic with arrows showing flow

Option 3: Video Frame Extraction
- Record combo execution
- Export key frames from video
- Combine into single visual breakdown

Specifications:
- 1280×720px minimum (wider is better for full-screen combos)
- Clear visibility of both characters
- Readable UI/damage numbers if visible
- Sequential order should be logical (left to right, top to bottom)

Testing in Guide:
After adding images, compile with:
  pdflatex Vishit.tex

Check PDF output to verify images integrate well with combo text.

See ../IMAGES_GUIDE.md for complete specifications and troubleshooting.
