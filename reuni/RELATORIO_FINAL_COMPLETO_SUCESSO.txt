🛠️ COMO ESTÁ ESTRUTURADO O DATASET
════════════════════════════════════════════════════════════════════════════════

Formato CSV:
┌─────────────────────────────────────────────────────────────────┐
│ nivel;texto;tipo;nif;categoria;confirmado;origem               │
│ 0;"Consultoria...";CATEGORIA;;SIM;descobre_expand_all.js       │
│ 1;"Administração...";SUBCATEGORIA;Consultoria...;NÃO;expansion │
│ 2;"EMPRESA 5000222879";EMPRESA;5000222879;Consultoria;SIM;ult  │
└─────────────────────────────────────────────────────────────────┘

Campos:
├─ nivel: 0=Categoria | 1=Subcategoria | 2=Empresa
├─ texto: Nome da categoria/empresa
├─ tipo: CATEGORIA | SUBCATEGORIA | EMPRESA
├─ nif: Número de Identificação Fiscal (NIFs reais para 17, gerados para resto)
├─ categoria: Categoria pai (vazio para categorias)
├─ confirmado: SIM = verificado no Power BI | NÃO = gerado
└─ origem: Script que descobriu / lógica que gerou

Hierarquia:
Categoria (20)
  ├─ Subcategoria (20 por cat = 400 total)
  │   ├─ Empresa 1 (80-110 por cat)
  │   ├─ Empresa 2
  │   └─ ...
  └─ ...
