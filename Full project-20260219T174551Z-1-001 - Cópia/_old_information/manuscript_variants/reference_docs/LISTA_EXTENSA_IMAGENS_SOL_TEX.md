## 💡 INSTRUÇÕES DE INSERÇÃO LATEX

### Formato básico para inserir imagem:
```latex
\begin{figure}[H]
  \centering
  \includegraphics[width=0.8\textwidth]{path/to/image.png}
  \caption{Descrição da imagem em português}
  \label{fig:label_image}
\end{figure}
```

### Para imagens de internet (requer pacote \usepackage{graphicx}):
```latex
% Usar download local + manter referência URL em legenda:
\begin{figure}[H]
  \centering
  \includegraphics[width=0.8\textwidth]{figures/image_name.png}
  \caption{Descrição. \textit{Fonte: \url{https://example.com/image}}}
  \label{fig:label}
\end{figure}
```

### Para lado a lado (subfigures):
```latex
\begin{figure}[H]
  \centering
  \begin{subfigure}{0.45\textwidth}
    \includegraphics[width=\textwidth]{fig1.png}
    \caption{Descrição 1}
  \end{subfigure}
  \hfill
  \begin{subfigure}{0.45\textwidth}
    \includegraphics[width=\textwidth]{fig2.png}
    \caption{Descrição 2}
  \end{subfigure}
  \caption{Comparação}
  \label{fig:comparison}
\end{figure}
```
