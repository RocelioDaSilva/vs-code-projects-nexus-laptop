# Manuscript Variants Archive

**Canonical manuscript:** `manuscript_unified/science_manuscript/manuscript/SOL.tex`  
**Canonical bibliography:** `manuscript_unified/science_manuscript/manuscript/referencias.bib`

Everything in this folder is a **non-canonical variant** of the manuscript — kept for historical reference only.

## Subfolders

### `backups/`
Timestamped backup copies created during editing sessions:
- `SOL.tex.backup` — Generic backup
- `SOL.tex.backup3` — Third numbered backup
- `SOL.tex.backup_before_fix` — Before a specific fix was applied
- `SOL.tex.backup_before_newline_fix` — Before newline corrections
- `SOL_BACKUP_20260305_234021.tex` — March 5, 2026 snapshot
- `SOL_melhorado_BACKUP_20260315_172151.tex` — March 15, 2026 snapshot of the "improved" version

### `experimental/`
Modified versions created during improvement attempts:
- `SOL_melhorado.tex` — "Improved" version with enhanced content (March 2026)
- `SOL_melhorado_FIXED_STRUCTURE.tex` — Structure-corrected variant
- `SOLEX.tex` — Extended/experimental version (purpose unclear)
- `SOL_SUBMISSION.tex` — Version formatted for journal submission (regenerate from canonical before actual submission)

### `old_manuscripts/`
The earliest versions of the manuscript:
- `papier.tex` — Original French/early draft
- `SOL.tex` — First version of the Solar manuscript
- `SOL.bbl` — Compiled bibliography
- `SOL_backup_20260207_230841.tex` — February 2026 backup
- `SOLV2IMPROVBYPT.TEX` — V2 improved by Portuguese team
- `Referências` — Early reference list
- `referencias.bib` — Original bibliography
- `figuras/` — LaTeX figure source files (maps)

### `reference_docs/`
Academic source lists and enhancement guides used during writing:

- Scientific standards and argumentation guides
- Reorganization reports
