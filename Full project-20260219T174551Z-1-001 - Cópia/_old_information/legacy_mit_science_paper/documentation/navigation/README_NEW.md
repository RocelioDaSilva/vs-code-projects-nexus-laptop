## ⚡ 60 Segundos: O que é GEESP?

| | |
|---|---|
| **O quê** | Sistema integrado de eletrificação solar para 5 comunidades remotas |
| **Onde** | Angola (Cacula, Chincua, Cuanhala, Dongorongo, Xancomingue) |
| **Quem** | MIT + ISPTEC + Governo Angola |
| **Objetivo** | Energia limpa, acessível, sustentável para famílias remotas |
| **Status** | Fase 1: 72% completo (Go-decision 15 Feb) |
| **Próximo** | Fase 2: Implantação no terreno (Feb-Mar) |
