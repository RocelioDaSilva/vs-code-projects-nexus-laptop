## 🎯 OBJECTIVOS DE RECOLHA DE DADOS  

### **Dados Técnicos (Radiação Solar)**
- Irradiação Global Horizontal (GHI): 5 dias contínuos, 1 medição/10min
- Irradiação Direta Normal (DNI) - se possível
- Temperatura ambiente, velocidade vento (dados complementares)
- **Equipamento:** Piranometer (classe A), datalogger, suporte estrutura

### **Dados de Infraestrutura & Localização**
- Pontos GPS: 5-8 locais por zona (igrejas, escolas, mercado, casarões, etc.)
- Precisão: CM-level (GPS diferencial, não apenas smartphone)
- Fotos geo-marcadas de cada localização (timestamp, coordenadas)
- Identificação de possíveis sítios mini-rede (superfície plana >500m², acesso veículos)

### **Dados Socioeconômicos (Surveys Comunitários)**
- Entrevistas estruturadas: 30-50 residentes (população sampling ~1%)
- Tópicos: Acesso energia, uso iluminação noturna, prioridades desenvolvimento
- Formulário padronizado (português, ~15 min/entrevista)
- Captura de contactos (possível comunidade focal group depois)

### **Dados Ambientais (Observação)**
- Cobertura vegetação (NDVI proxy via visual)
- Condições topografia (encostas, vales, platôs)
- Uso do solo (agrícola, pastagem, urbano, floresta)
- Observação erosão, contaminação, biodiversidade

```
        - Seleção de 8 pontos GPS (distribuição espacial ~2km²)
16:00 - Instalação estrutura piranometer

er setup + teste comunicação
        - Calibração inicial (verificar sensor response)
```

#### **Dia 2-4 (Ter-Qua 19-21 mai): Medições & Coleta Dados**

```
05:30 - Acordar; café
06:00 - INÍCIO Medições Solares
        - Piranometer: recolher dados 24h antigos (backup)
        - Verificar status: voltagem, conectividade, erros
        - Limpeza diária lente piranometer (pó, insetos)
06:30 - Equipa A: Pontos GPS
        - Técnico 1 + Técnico 2: Viajam para 2-3 locais GPS identificados
        - Precisão: ±5cm esperado com GPS diferencial
        - Fotografia geo-marcada de cada ponto + reference object
        - Documentação: nomes locais, landmarks, access easiness
        - Tempo por ponto: ~20 min = 3h total
10:30 - Retorno base; análise qualidade dados GPS
11:00 - Entrevistas Comunitárias (Parte 1)
        - Formário aberto com 15-20 residentes
        - Locais: igreja, mercado, escola
        - Tópicos: acesso energia, uso iluminação, prioridades (saúde, educação, água)
        - Entrevistador: fala português Angola
        - Tempo: 15 min/entrevista ×15 = 225 min = 4h com logistic
12:30 - Almoço
14:00 - Continuação entrevistas comunitárias (Parte 2)
        - Mais 10-15 residentes
        - Cobrir diferentes grupos etários, géneros
15:00 - Mapeamento sítios possíveis mini-rede
        - Procurar localização com potencial
        - Critérios: área plana >500m², acesso veículos, proximidade população
        - GPS de 2-3 canditatos
16:00 - Repouso/folga equipa
17:00 - Análise dados do dia
        - Descarregar GPS para laptop
        - Verificar qualidade fotos + gerar lista de erros
18:30 - Ceia
20:00 - Repouso

[Repete Dia 3-4 com variação de localizações e residentes]
```

#### **Dia 5 (Qui 22 mai): Finalização & Partida**

```
06:00 - Verificação final piranometer (download últimos dados 24h)
07:00 - Desmontagem estrutura + GPS pacote para transporte
08:00 - Reunião comunitária resumo
```

## 🆘 PROTOCOLO DE EMERGÊNCIA

### **Cenários & Respostas**

1. **Falha Piranometer (impossível registar radiação):**
   - Ativar backup equipment (aluguel 2ª device) - 1-2 dias
   - Usar dados Sentinel-2 como proxy temporário
   - Estender período de recolha em 3-5 dias

2. **Atraso Transporte / Impossibilidade Acesso (estrada bloqueada, enchente):**
   - Plano alternativo: usar 4WD ou motor-bike transporte local
   - Agendar voo charter if needed (USD 2K+)
   - Estender campanha +2 dias se necessário

3. **Doença Equipa / Hospitalização:**
   - Contactar MIT Advisor immediately
   - Evacuação médica via seguro travel (pré-arranjo)
   - Substituição equipa ( backup técnico pre-identificado)

4. **Rejeição Comunitária / Recusa Consentimento:**
   - Pausar atividades; dialogar com líderes
   - Voltar Luanda; re-comunicação &clarificação
   - Possível mudança zona (backup: Malanje province)

5. **Segurança: Ameaça / Conflito Local:**
   - Evacuação imediata
   - Contactar embaixada MIT (+ embaixada USA)
   - Escalação para Rocélio + MIT President
