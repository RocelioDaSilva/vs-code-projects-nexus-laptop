# GEESP-Angola Frontend

React + Vite frontend for the Geospatial Energy for Equity & Solar Planning (GEESP) application.

## Prerequisites

- Node.js (v18+)
- Backend API running on port 8000

## Setup

1. Copy `.env.example` to `.env.local` and configure environment variables
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`

## Scripts

- `npm run dev` — Start both client and API server in development mode
- `npm run build` — Production build
- `npm run lint` — TypeScript type checking
- `npm run preview` — Preview production build
