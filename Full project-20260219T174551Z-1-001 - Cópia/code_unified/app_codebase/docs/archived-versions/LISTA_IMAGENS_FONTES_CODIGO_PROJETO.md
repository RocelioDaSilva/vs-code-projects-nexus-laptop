## 💾 COMO USAR ESTAS IMAGENS

### Download em Batch
```bash
# Criar pasta de destino
mkdir -p projeto/02_Code/assets/{logos,diagrams,screenshots}

# Fazer download de imagens (usando wget ou curl)
# Exemplo:
wget https://commons.wikimedia.org/wiki/File:React-icon.svg -O projeto/02_Code/assets/logos/react.svg
```

### Integração em Documentação
```markdown
# Exemplo para README.md
![React](assets/logos/react.svg) ![TypeScript](assets/logos/typescript.svg) ![Express](assets/logos/express.svg)

## Architecture
![System Architecture](assets/diagrams/architecture.png)
```

### Usar em Código React
```jsx
import { ReactComponent as SolarIcon } from '@/assets/icons/solar.svg';

export function DashboardHeader() {
  return (
    <>
      <SolarIcon className="w-8 h-8 text-yellow-500" />
      <h1>GEESP-Angola Dashboard</h1>
    </>
  );
}
```
