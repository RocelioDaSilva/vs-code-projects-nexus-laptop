## How to Use This Directory

**For writing/revision:**
1. Identify what information you need
2. Use Quick Navigation above to find document
3. Reference the specific section within document
4. Update document if you change how something works

**For research:**
1. Browse appropriate subdirectory README
2. Follow cross-references to related documents
3. Use search function (Ctrl+F) within documents

**For consolidation:**
1. Each subdirectory has README explaining its contents
2. "MASTER" files consolidate related information
3. See `documentation/REORGANIZATION-PLAN.md` for consolidation strategy
