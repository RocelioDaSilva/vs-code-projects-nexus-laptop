## How to Use This Index

### For Writers

**Writing a chapter?**
1. Find chapter in [All Chapters list](#all-chapters-full-list)
2. Check status (Draft, Outline, Not Started)
3. Check word target
4. Search `story/CHAPTER-BY-CHAPTER/` for the file
5. Reference scenes from appropriate `SCENES-BY-ARC/` folder

**Writing a scene?**
1. Identify topic (combat, emotion, worldbuilding, etc.)
2. Check [Scenes by Theme](#scenes-by-theme)
3. Look for similar completed scenes in that category
4. Place new scene in appropriate `SCENES-BY-ARC/` subfolder
5. Update this index with new entry

**Researching continuity?**
1. Find character name or moment
2. Cross-reference [Character Index](../characters/CHARACTER-MASTER-INDEX.md)
3. Check related scenes in [Scenes by Theme](#scenes-by-theme)
4. Verify timeline using chapter ages (Aisen Age column)

### For Status Tracking

**Update frequency**: After each writing session
**Location**: Update this file + `CHAPTER-STATUS-TRACKER.md`
**Status values**:
- NOT STARTED (0%) — No draft yet
- OUTLINE (10%) — Plot outline exists
- DRAFT (90%) — Full rough draft done
- REVISION (95%) — Editing in progress
- FINAL (100%) — Ready for publication
