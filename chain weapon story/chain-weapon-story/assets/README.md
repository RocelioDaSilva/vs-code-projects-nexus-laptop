# Assets Directory

This folder stores all visual assets, images, graphics, and media files for the Chain Weapon Story project.

## Purpose

Organize and store:
- **Character artwork** (visual representations of characters)
- **Map imagery** (world maps, regional maps, layout diagrams)
- **Concept art** (worldbuilding visual references)
- **Diagrams & charts** (magic system flowcharts, power visualizations)
- **Cover art** (if creating book cover)
- **Reference images** (visual inspiration from research)

## Folder Structure

```
assets/
├── images/                          [Visual files go here]
│   ├── characters/                 [Character artwork]
│   ├── maps/                       [World maps]
│   ├── concepts/                   [Concept art]
│   ├── diagrams/                   [Technical diagrams]
│   └── reference/                  [Reference imagery]
└── .gitkeep                        [Folder preservation marker]
```

## File Naming Convention

**For consistency, use:**
- Format: `[TYPE]-[SUBJECT]-[VERSION].ext`
- Examples:
  - `map-shukei-nation-v1.png`
  - `character-aisen-portrait-v2.jpg`
  - `diagram-magic-system-flux-v1.pdf`
  - `concept-academy-architecture-v1.png`

**Acceptable formats:**
- Images: `.png`, `.jpg`, `.jpeg`, `.gif`, `.webp`
- Diagrams: `.pdf`, `.svg`, `.png`
- Vector: `.svg` (recommended for scalable graphics)

Assets folder is prepared for visual media storage as project develops.
