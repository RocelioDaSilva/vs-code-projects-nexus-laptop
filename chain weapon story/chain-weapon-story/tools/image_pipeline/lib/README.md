ImageTracerJS

This folder is for the `imagetracer_v1.2.6.js` library if you prefer a local copy instead of installing the npm package.

Download `imagetracer_v1.2.6.js` from https://github.com/jankovicsandras/imagetracerjs and place it here.

Usage options:
- `npm install imagetracerjs` (preferred) OR
- download `imagetracer_v1.2.6.js` and put it in this folder (the `generate.js` will attempt to load it).
