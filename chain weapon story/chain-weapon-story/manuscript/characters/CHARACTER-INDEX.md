## How to Use During Writing

### For Chapter Setup
1. **Identify which characters appear**: Check relevant profile
2. **Review their relationship**: See relationship map section
3. **Check voice consistency**: Use voice notes during dialogue scenes
4. **Verify physical traits**: Describe using persistent traits from profile

### For Scene Description
1. **Physical appearance**: Use description section (not just "he walked in")
2. **Distinctive mannerisms**: Reference their habitual gestures
3. **Internal consistency**: Check they act according to psychology profile
4. **Thematic resonance**: Verify scene advances their character arc

### For Dialogue
1. **Voice patterns**: Reference dialogue/voice notes
2. **Speech tags**: Use only when necessary (voice should distinguish)
3. **Character voice variation**: Check accents, formality, metaphor types
4. **Emotional state**: Verify voice changes when emotional

### For Conflict/Tension
1. **Relationship dynamics**: Check relationship map
2. **Character motivation**: Reference core drive section
3. **Internal paradox**: Use as source of conflict/tension
4. **Growth arc**: Verify conflict moves them along arc

## Post-Writing Verification Checklist

After writing scenes with multiple characters:
- [ ] Verify each character's age is consistent with timeline
- [ ] Check physical traits match profile (scars, markings, etc.)
- [ ] Voice consistency: Each character sounds distinct
- [ ] Relationship authenticity: Interactions match relationship map
- [ ] Emotional authenticity: Characters act according to psychology
- [ ] Thematic function: Scene advances character's arc
- [ ] Mannerisms included: 2-3 distinctive behaviors per character
- [ ] Callbacks to past: Acknowledge shared history where relevant
