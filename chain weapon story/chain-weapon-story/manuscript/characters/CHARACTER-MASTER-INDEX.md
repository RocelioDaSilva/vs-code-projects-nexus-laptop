## CHARACTER TIER DEFINITIONS

### TIER 1: PRIMARY
- **POV Scenes**: 5+ chapters from this character's perspective (or will be)
- **Story Impact**: Directly drives major plot events or theme development
- **Development**: Full arc from introduction to climax/resolution
- **Relationships**: Core connections to 3+ other characters
- **Profile**: 2000+ words, detailed voice, background, motivations, conflicts

**Examples**: Aisen (protagonist), Caspian (tragic opposite), Seraphine (antagonist force)

### TIER 2: SECONDARY  
- **POV Scenes**: 1-3 chapters from this character's perspective (or features in 5+ scenes)
- **Story Impact**: Supports protagonist arc or enables major plot events
- **Development**: Character arc present but secondary to protagonist
- **Relationships**: Strong connections to protagonist or other primaries
- **Profile**: 1000-2000 words, established voice, clear motivations, defined arc

**Examples**: Harald (mentor), Elara (ally), Lysandra (resistance contact)

### TIER 3: ENSEMBLE
- **POV Scenes**: 0-1 dedicated scene (may appear in multiple chapters)
- **Story Impact**: Provides context, challenge, or thematic resonance
- **Development**: Static or minimal change; role is defined
- **Relationships**: Knows protagonist or one primary character well
- **Profile**: 300-800 words, distinct trait or ability, clear function, memorable detail

**Examples**: Bloodheart (specialized power), Kairos (encounter/challenge), Lira (microscene focus)

## HOW TO USE THIS INDEX

1. **Finding a character**: Search this document for name; see file location and tier
2. **Understanding status**: Check "Status" column; if "Needs Expansion," use as todo item
3. **Adding a new character**: 
   - Determine tier based on definitions above
   - Create profile file in appropriate tier folder
   - Add row to this index
   - Link from story file(s) where character appears
4. **Consolidating duplicates**: Archive old version; update this index with new canonical location
5. **Starting to write a chapter**: Reference relevant characters and their last appearance/motivation in this index
