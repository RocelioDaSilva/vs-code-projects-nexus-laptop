Sanderson Checklist:
[ ] Did I show a cost for every magical act?
[ ] Did I use a limitation to create a problem?
[ ] Did I expand an existing ability instead of adding a new one?

Abercrombie Checklist:
[ ] Did every character have a selfish moment?
[ ] Did doing the "right thing" make things worse?
[ ] Did I include dark humor in a bleak scene?

Jemisin Checklist:
[ ] Did the environment affect the scene?
[ ] Did class or systemic oppression create an obstacle?
[ ] Did I include a marginalized perspective?

Rothfuss Checklist:
[ ] Did I use at least three senses in the scene?
[ ] Did I plant a mystery to be resolved later?
[ ] Did I use a recurring motif?

Martin Checklist:
[ ] Did I show multiple factions with competing goals?
[ ] Did a leader make a hard, cruel choice?
[ ] Did information (letter, spy, rumor) drive the plot?

Hobb Checklist:
[ ] Did I live inside the character's head?
[ ] Did past trauma affect present choices?
[ ] Did the character fail or relapse?
