## ✍️ Writing Workflow

1. **Check CHAPTER-STATUS-TRACKER.md** to see which chapters need work
2. **Reference story-overview.md** for protagonist arc and systems overview
3. **Use CHARACTER-MASTER-INDEX.md** for character voice and relationships
4. **Follow CONTRIBUTING.md** for file formatting and naming standards
5. **Write chapter in story/ folder**, maintaining frontmatter metadata
6. **Update chapter status** in tracker when done

## 📝 Writing Process

1. Choose scene from **structure/SCENE-TESTS-CHAPTERS-4-7.md**
2. Review Multi-PoV breakdown and subplots
3. Follow opening hook guidance
4. Write 2000–4000 words (check target in scene test)
5. Validate against scene function and closing moment
6. Move to next scene
