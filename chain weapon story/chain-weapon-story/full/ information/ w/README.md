# Full Information Directory

This folder stores comprehensive information compilations, extended reference notes, and full documentation pages that synthesize information across multiple sources.

## Purpose

- **Comprehensive notes** (full synthesis of topic information)
- **Extended references** (longer-form reference material)
- **Information compilations** (gathering related info from multiple sources)
- **Detailed frameworks** (complete system documentation)
- **Reference synthesis** (combining related documents into master references)

## Content Organization

```
full-information/
├── README.md                       [This file]
├── [Topic]-COMPREHENSIVE.md        [Full information on specific topics]
├── [Topic]-SYNTHESIS.md            [Information synthesis files]
└── [Other comprehensive docs]      [Additional full information files]
```

## Types of Documents Stored Here

1. **Comprehensive Compilations** — Full information on single topics
   - Example: `Magic-System-Comprehensive.md`
   - Used for: Complete reference without splitting across files

2. **Information Synthesis** — Combining related information from multiple sources
   - Example: `World-Information-Synthesis.md`
   - Used for: Creating master documents from distributed information

3. **Extended References** — Longer-form reference material
   - Example: `Character-Relationship-Comprehensive.md`
   - Used for: Deep dives into specific topics

4. **Framework Documentation** — Complete system documentation
   - Example: `Magic-System-Framework-Complete.md`
   - Used for: Understanding how systems work in full detail

## Naming Convention

For files in this folder, use:
- Format: `[TOPIC]-COMPREHENSIVE.md` or `[TOPIC]-SYNTHESIS.md`
- Examples:
  - `Magic-System-Comprehensive.md`
  - `World-Information-Synthesis.md`
  - `Character-Relationships-Comprehensive.md`
  - `Power-Systems-Full-Documentation.md`

## Current Status

This folder is prepared for comprehensive information storage and synthesis work as documentation expands.

## When to Use This Folder

Use this folder for documents that:
- [ ] Cover a complete topic in great detail
- [ ] Synthesize information from 5+ other sources
- [ ] Provide comprehensive reference material
- [ ] Are too long for quick-reference files
