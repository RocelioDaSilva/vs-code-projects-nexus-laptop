## 5. ESTRUTURA LaTeX RECOMENDADA

### **Configuração de Documentclass**
```latex
\documentclass[12pt,twoside]{report}  % Report para documentos longos
\usepackage[utf8]{inputenc}
\usepackage[portuguese]{babel}
\usepackage[margin=1in]{geometry}
\usepackage{hyperref}
\usepackage{graphicx}
\usepackage{fancyhdr}
\usepackage{titlesec}
\usepackage{tocloft}
\usepackage{longtable}
\usepackage{listings}
\usepackage{xcolor}
```

### **Componentes Essenciais**
1. **Table of Contents** (Índice Geral)
2. **List of Figures** (Índice de Figuras)
3. **List of Tables** (Índice de Tabelas)
4. **Executive Summary** (1-2 páginas)
5. **Body** com 17 capítulos principais
6. **Appendices** (A-H)
7. **Bibliography** (Normas científicas)

### **Estilos Recomendados**
- **Títulos (Chapter):** \Large\bfseries\color{darkblue}
- **Subsecções:** \large\bfseries
- **Tabelas:** use `longtable` para flexibilidade
- **Figuras:** Incluir captions descritivos
- **Diagramas:** usar TikZ para organogramas
