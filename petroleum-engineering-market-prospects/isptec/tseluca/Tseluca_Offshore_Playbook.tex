\chapter*{How To Use This Book}
This book is an execution manual for one goal: get into a petroleum-focused offshore rotational role quickly, with a strong salary path and low unemployment risk after graduation.

Use this method:
\begin{enumerate}
  \item Read Chapters 1 to 3 in one sitting.
  \item Build your personal checklist from Chapters 4 to 7.
  \item Execute weekly using Chapter 8 scorecards.
  \item Review and update your plan every Sunday.
\end{enumerate}

\chapter{Your Goal and Best Strategy}

\section{Your stated target}
\begin{itemize}
  \item Work offshore in FIFO rotation lifestyle.
  \item Earn good salary with upward trajectory.
  \item Avoid long unemployment after graduation.
  \item Stay in petroleum-engineering-focused roles.
\end{itemize}

\section{Best combined route from repository guidance}
\begin{enumerate}
  \item Pick one primary technical path: Drilling or Production/FPSO.
  \item Complete mobilization certifications early.
  \item Apply in waves to multiple companies in parallel.
  \item Enter via high-employability routes first (service companies, then operators/FPSO).
  \item Build technical depth and move up by proven performance.
\end{enumerate}

\section{Career architecture}
\begin{itemize}
  \item Stage 0: final year and graduation launch.
  \item Stage 1: first role (0 to 6 months after graduation).
  \item Stage 2: specialization growth (Years 1 to 2).
  \item Stage 3: salary acceleration (Years 2 to 5).
  \item Stage 4: premium offshore technical roles (Year 5+).
\end{itemize}

\chapter{What To Do First: 90-Day Launch Plan}

\section{Days 1 to 15}
\begin{itemize}
  \item Choose primary path: Drilling or Production/FPSO.
  \item Choose backup path: HSE-support or Completions-support.
  \item Build target company sheet (minimum 20 companies).
  \item Open certification savings plan and fund it monthly.
  \item Join SPE student membership.
\end{itemize}

\section{Days 16 to 45}
\begin{itemize}
  \item Complete offshore medical.
  \item Book OPITO BOSIET with HUET.
  \item Start IWCF Level 1 preparation.
  \item Build CV and LinkedIn profile (Portuguese and English ready).
\end{itemize}

\section{Days 46 to 75}
\begin{itemize}
  \item Complete BOSIET/HUET.
  \item Complete H2S awareness (if required by targets).
  \item Complete IWCF Level 1.
  \item Build first technical mini-project (Excel or Python).
\end{itemize}

\section{Days 76 to 90}
\begin{itemize}
  \item Submit first application wave (10 to 15 strong applications).
  \item Run mock interviews weekly.
  \item Attend one SPE/ANPG/company networking event.
  \item Start NEBOSH IGC if budget and role strategy support it.
\end{itemize}

\section{Anti-scam quality checks before payment}
\begin{itemize}
  \item Verify accreditation on the official awarding body website.
  \item Confirm exact certificate title and validity in writing.
  \item Confirm target employer acceptance before payment.
  \item Confirm all fees (tuition, exam, practical, re-sit, travel).
  \item Review cancellation and refund policy.
\end{itemize}

\chapter{Interview and Offer Strategy}

\section{Technical interview preparation}
Focus areas:
\begin{itemize}
  \item Well control basics and pressure logic.
  \item Drilling fluid fundamentals and ECD reasoning.
  \item Production basics: IPR/VLP concepts and troubleshooting thinking.
  \item Offshore safety behavior and incident prevention mindset.
\end{itemize}

\section{Behavioral interview preparation}
\begin{itemize}
  \item Teamwork under fatigue and shift pressure.
  \item Communication in multicultural offshore crews.
  \item Safety-first decisions when production pressure exists.
  \item Learning speed and humility in field operations.
\end{itemize}

\section{Offer evaluation checklist}
\begin{itemize}
  \item Rotation pattern and on/off days.
  \item Base pay + offshore allowances + total package.
  \item Certification sponsorship and training roadmap.
  \item Scope of technical work and growth path.
  \item Team quality and mentorship access.
\end{itemize}

\chapter{Weekly Operating System}

\section{Weekly scorecard}
Score each item from 0 to 1.
\begin{itemize}
  \item 6 to 8 hours technical study completed.
  \item 2 problem sets completed.
  \item 3 networking actions completed.
  \item 5 quality applications or 2 referral actions completed.
  \item 1 interview practice completed.
  \item 1 certification savings transfer completed.
\end{itemize}

Interpretation:
\begin{itemize}
  \item 5/6 = strong week.
  \item 6/6 = elite week.
\end{itemize}

\section{Monthly review}
\begin{enumerate}
  \item Which actions produced interviews?
  \item Which gaps blocked progress?
  \item Which certificate or skill must move up in priority?
  \item What is the next 30-day objective?
\end{enumerate}
