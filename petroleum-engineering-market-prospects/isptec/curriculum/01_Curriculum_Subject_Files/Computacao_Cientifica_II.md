## What You Learned

Programming in MATLAB or Python (or both), numerical methods implementation, data visualization, algorithm design, and applying computational tools to solve engineering problems that cannot be solved by hand.

## Where This Subject Is Used in Industry

- **Python for Petroleum Engineering:** Python is rapidly becoming the standard tool in petroleum engineering. Reservoir engineers use Python for decline curve analysis, data cleaning, statistical analysis, and machine learning. Production engineers use Python to analyze production data, automate reports, and build dashboards.
- **MATLAB:** Some companies (especially in research and reservoir simulation) use MATLAB for numerical computation, visualization, and prototyping algorithms.
- **Automation:** Repetitive tasks that take hours in Excel (processing well test data, generating production allocation reports, formatting daily reports) can be automated in Python in minutes.
- **Data Science in Oil & Gas:** The industry is increasingly using machine learning for predictive maintenance (ESP failure prediction), production optimization, and automated log interpretation. All of this requires programming skills.

## Angola-Specific Context

Major operators in Angola ([TotalEnergies](../../../data/company-directory/totalenergies.md), Eni) are investing in digitalization:
- **Digital twins** of FPSOs require software engineering
- **Production optimization** using real-time data analytics requires Python
- **Automated reporting** to ANPG uses scripted data workflows

A petroleum engineer who can code in Python is significantly more valuable than one who cannot. This is the most directly employable "non-petroleum" skill in the curriculum.
