Inspection Checklist (Streamlit)

Objetivo
- Criar um checklist digital simples para inspeções com fotos, comentários e status.

Pré-requisitos
- `pip install -r ../requirements.txt`
- `pip install streamlit`

Passos
1. Rodar `streamlit run checklist.py`
2. Usar campos para preencher itens, marcar OK/NOK, anexar foto
3. Exportar CSV do relatório de inspeção

Arquivo principal: `checklist.py`
