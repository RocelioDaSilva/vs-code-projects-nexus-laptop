Volve Data Analysis

Objetivo
- Exemplo de EDA e replicação de análises com o dataset Volve (demo) usando pandas/plotly.

Pré-requisitos
- `pip install -r ../requirements.txt`

Passos
1. Baixar dados Volve (links públicos) e colocar em `data/`
2. Rodar `python volve_eda.py` ou abrir `notebooks/volve_eda.ipynb`

Arquivo principal: `volve_eda.py`
