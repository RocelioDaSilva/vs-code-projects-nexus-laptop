## 2️⃣ TIPOS DISPONÍVEIS

### ANTES (6 tipos):
```
• Convencional Onshore
• Petróleo Pesado/Viscoso
• Profundo/Ultra-profundo
• Offshore Intermediário
• Carbonatado
• Teor Argila Elevado
```

### DEPOIS (7 tipos):
```
🎯 SEM SELEÇÃO ESPECÍFICA    ← [NOVO]
   └─ Para análise genérica sem restrições geológicas

• Convencional Onshore
  ├─ Composição: Arenito com argilito
  ├─ Desafios: 4 documentados
  └─ Métodos: Polímero, Surfactante, Alcalino, CO₂

• Petróleo Pesado/Viscoso
  ├─ Composição: Arenito/argilito com petróleo viscoso
  ├─ Desafios: 7 documentados
  └─ Métodos: Vapor, Combustão, CO₂

• Profundo/Ultra-profundo
  ├─ Composição: Arenito compactado (P e T elevadas)
  ├─ Desafios: 7 documentados
  └─ Métodos: CO₂ Miscível, N₂ Miscível

• Offshore Intermediário
  ├─ Composição: Arenito com siltito e argilito
  ├─ Desafios: 7 documentados
  └─ Métodos: CO₂ Miscível, Polímero, WAG

• Carbonatado ⭐ MAIS DETALHADO
  ├─ Composição: Calcário/dolomita com anidrita/gesso
  ├─ Desafios: 8 documentados (máximo)
  └─ Métodos: CO₂ Miscível, Gás HC, WAG

• Teor Argila Elevado ⭐ MAIS DETALHADO
  ├─ Composição: Areia + >10% argila (Ilita, Caolinita, Montmorillonita)
  ├─ Desafios: 8 documentados (máximo)
  └─ Métodos: CO₂ Miscível, Vapor, WAG
```

## 3️⃣ INFORMAÇÕES POR TIPO

### EXEMPLO: CARBONATADO

#### ANTES:
```
Carbonatado
Profundidade: Variável (500-3500m)
API ideal: 18-35
Molhabilidade: Frequentemente hidrofóbica
Métodos prioritários: CO₂ Miscível, Gás HC, Métodos alternativos
Recuperação típica: 5-20%
Presença de anidrita/gesso consome alcalino. Gás miscível preferível.
```

#### DEPOIS:
```
╔════════════════════════════════════════════════════════════════════╗
║  TIPO DE RESERVATÓRIO: Carbonatado                                ║
╚════════════════════════════════════════════════════════════════════╝

━━━ CARACTERÍSTICAS GEOLÓGICAS ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Profundidade:        Variável (500-3500m)
API Ideal:           18-35
Viscosidade:         Variável
Composição:          Calcário ou dolomita com possível anidrita/gesso.
                     Molhabilidade frequentemente hidrofóbica (oil-wet)
Porosidade:          5-20%
Permeabilidade:      Controlada por fraturas naturais

━━━ MÉTODOS EOR RECOMENDADOS ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✓ Prioritários:      CO₂ Miscível, Gás Hidrocarbonetos, WAG
○ Secundários:       Polímero - efetividade reduzida
✗ A Evitar:          Alcalino - consumo excessivo, Surfactante

━━━ ANÁLISE E CARACTERIZAÇÃO ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Presença de anidrita/gesso consome (reage com) alcalino. Molhabilidade
hidrofóbica reduz eficiência de água. Gás miscível preferível.

Recuperação Esperada: 5-20%
Custo Relativo:      Muito Alto

━━━ DESAFIOS TÉCNICOS E RISCOS ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. Molhabilidade hidrofóbica reduz imbibição de água
  2. Anidrita/gesso consome (reage com) alcalino e CO₂
  3. Fraturas naturais podem causar vazamento prematuro de injetados
  4. Permeabilidade dependente de tensão efetiva - mudanças de 
     pressão afetam fluxo
  5. Heterogeneidade extrema dificulta previsão de desempenho
  6. Possível geração de finos (calcita precipitada) bloqueando poros
  7. pH sensível em calcário - controle crítico necessário
  8. Eficiência de varrido comprometida por contraste de mobilidade

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
