### Até 3 linhas
```

Ou: "Os deveres da conduta..." (CASTRO, 2001, p. 23)
```

### Mais de 3 linhas
```
    Texto da citação direta com mais de três 
    linhas deve vir destacado, com recuo de 
    4cm à esquerda, fonte menor (10 ou 11pt),
    sem aspas, espaçamento 1cm entre linhas
    e alinhamento justificado. (AUTOR, ano, p. XX)
```

## 📌 DICAS IMPORTANTES

### Para Objetivos Específicos
- Use verbos no infinitivo
- Exemplo: caracterizar, identificar, compreender, analisar, verificar
- Formato em lista com letras após ponto e vírgula

### Para Resumo
- Mínimo: 150 palavras
- Máximo: 500 palavras
- Parágrafo único
- Deve conter 3 palavras-chave

### Para Abstract
- Tradução do resumo para inglês
- 3 Key-words em inglês
- Mesma estrutura do resumo

### Para Tabelas/Figuras
- Colocar próximo ao texto que menciona
- Fonte OBRIGATÓRIA, mesmo se elaborado pelo autor
- Numeração consecutiva
- Títulos acima, fonte abaixo

### Para Notas de Rodapé
- Numeração consecutiva ao longo do texto
- Após ponto e vírgula: letra minúscula
