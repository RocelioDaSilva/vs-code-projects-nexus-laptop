## Exemplo de Uso Rápido

### Matriz
```
1. Clique "Todos"
2. Desmarque alguns métodos
3. Clique "Gerar Matriz"
4. Selecione um método no dropdown
5. Veja características na caixa azul
```

### Comparativo
```
1. Selecione 2-3 métodos
2. Escolha tipo "Radar"
3. Clique "Métodos Selecionados"
4. Veja comparação visual
```

### Resultados
```
1. Selecione tipo "Dashboard"
2. Clique "Gerar Gráfico"
3. Veja KPIs + 4 gráficos
4. Clique "Exportar Gráfico"
5. Escolha PNG/PDF/SVG
```
