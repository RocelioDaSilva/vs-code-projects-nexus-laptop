# 04_Operations – DevOps & Infrastructure

This folder contains **operational** assets: scripts, infrastructure as code, Kubernetes, Ansible, and monitoring.

| Subfolder | Contents |
|-----------|----------|
| **scripts/** | DB backup/restore/init, cron, Docker tests, GitHub issue scripts, etc. |
| **infrastructure/** | Terraform, CloudFormation. |
| **kubernetes/** | K8s manifests (formerly k8s/). |
| **ansible/** | Ansible playbooks. |
| **monitoring/** | Prometheus, Grafana, Alertmanager configs. |

Use these when deploying, backing up, or operating the system.
