# 02_Code – Application Code

This folder contains all **application code** for GEESP-Angola.

| Subfolder | Contents |
|-----------|----------|
| **geesp-angola/** | Main Streamlit app, launchers, scripts, dashboard, monitoring, tests, data. |

**Run the app**
- From project root: double-click `run_geesp_app.bat`
- From here: `geesp-angola/launch_app.bat` (Windows) or `geesp-angola/launch_app.sh` (Linux/macOS)

**Entry point**: `geesp-angola/geesp_unified_app.py`

**Docs**: `geesp-angola/README.md`, `geesp-angola/docs/CAPABILITIES.md`, `geesp-angola/docs/IMPROVEMENTS.md`
